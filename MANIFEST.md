**Structure needed:**
```

├── index.html

├── CBS_W1_S3_Data_Governance.html
├── CBS_W1_S4_Quantum.html

        └── pages.yml
```

### Step 2: Follow GITHUB_SETUP.md

Open GITHUB_SETUP.md for complete step-by-step instructions on:
- Creating a GitHub repository
- Uploading your files
- Enabling GitHub Pages
